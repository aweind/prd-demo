Useepay
电子钱包支付（Apple Pay、Google Pay 等）提供快速、安全的结账体验，顾客可借助设备原生验证一键完成支付。
请在Apple开发者后台下载域名验证文件并在此上传完成验证。Google Pay 无需此步骤。
PRD 注释与业务规则

渠道与字段联动

  • 通用字段: Mch No, App ID, Publishable Key, Api Key 为各渠道通用。
  • Apple Pay专属字段: 域名验证文件, Merchant Id, Private/Public Key 等。若为 Google Pay 渠道，则前端会自动隐藏无关字段。

交互说明：操作与反馈

  • 下挂地址：不开启的情况下，url输入框置灰禁用。
  • 网关状态联动：未激活时仅右对齐可见【激活】按钮。已激活但无改动时显示【停用】按钮，有改动时额外显示【保存】按钮。操作触发吐司提示。
